# TeamFinder

Платформа для поиска единомышленников и совместной работы над pet-проектами.

**Вариант 2:** навыки пользователей и фильтрация участников по навыкам.

**Автор:** Белова Полина, РЭУ им. Г. В. Плеханова, 15.27Д-МО01/23б

---

## О проекте

TeamFinder — веб-платформа, где авторы публикуют идеи pet-проектов, а разработчики, дизайнеры, тестировщики и другие специалисты находят интересные проекты и присоединяются к ним.

## Функциональность

- Регистрация и вход по email и паролю
- Создание, редактирование и завершение проектов
- Участие в чужих проектах
- Личный профиль с редактированием (аватар, контакты, описание)
- Управление навыками (добавление и удаление)
- Фильтрация участников по навыкам
- Пагинация (12 проектов и 12 пользователей на страницу)
- Автоматическая генерация аватарки при регистрации
- Админ-панель для управления пользователями и проектами

## Технологии

- Python 3.13
- Django 5.2
- SQLite / PostgreSQL
- HTML, CSS, JavaScript
- Docker / Docker Compose
- Pillow — генерация аватаров

## Запуск

1. Клонируйте репозиторий и перейдите в папку проекта.
2. Установите зависимости:

```bash
pip install -r requirements.txt
```

3. Создайте файл окружения:

```bash
copy .env_example .env
```

4. Примените миграции и запустите сервер:

```bash
python manage.py migrate
python manage.py runserver
```

5. Откройте в браузере: http://127.0.0.1:8000/

Админ-панель: http://127.0.0.1:8000/admin/
